# 카페24 백엔드 서버 배포 대안

## ❌ 카페24 제약사항

카페24는 **PHP 기반 쇼핑몰 호스팅**이므로:
- ✅ 정적 파일 (HTML, CSS, JS, 이미지) - 가능
- ✅ PHP 스크립트 - 가능
- ❌ Python Flask 서버 - **불가능**

## ✅ 해결 방법

### 방법 1: 무료 클라우드 호스팅 사용 (추천)

**Render.com** (무료 플랜 있음)
- Python Flask 지원
- 자동 배포
- 무료 플랜: 750시간/월 (거의 24시간)

**Railway.app** (무료 크레딧)
- Python Flask 지원
- 간단한 배포

**PythonAnywhere** (무료 플랜)
- Python 전용 호스팅
- Flask 앱 직접 배포 가능

### 방법 2: 카페24 + 외부 API 조합

1. **프론트엔드**: 카페24 디자인보관함에 HTML 업로드
2. **백엔드**: Render/Railway 등에 Python 서버 배포
3. **연결**: HTML에서 외부 API URL로 요청

```javascript
// 카페24 HTML에서 사용
const REMOVE_BG_API_URL = 'https://your-app.onrender.com/api/remove_bg';
```

### 방법 3: 카페24 PHP로 래핑 (복잡함)

카페24에서 PHP로 Python 스크립트 호출 시도:
- `exec()` 또는 `shell_exec()` 사용
- 보안/권한 문제 가능
- 권장하지 않음

## 🚀 추천: Render.com 무료 배포

1. **GitHub에 코드 푸시** (이미 완료)
2. **Render.com 가입**
3. **New Web Service** → GitHub 연결
4. **설정**:
   - Build Command: `pip install -r requirements_image_backend.txt`
   - Start Command: `gunicorn image_bg_backend:app`
5. **배포 완료** → `https://your-app.onrender.com` 생성
6. **HTML에서 URL 변경**
