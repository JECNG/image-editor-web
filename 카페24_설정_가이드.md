# 카페24 디자인보관함에 TensorFlow.js 라이브러리 업로드 가이드

## 📦 업로드할 파일

### 1. TensorFlow.js Core
- **파일**: `tf.min.js`
- **다운로드**: https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.10.0/dist/tf.min.js
- **크기**: 약 1.5MB

### 2. BodyPix 모델 (선택사항 - 사람 전용)
- **파일**: `body-pix.min.js`
- **다운로드**: https://raw.githubusercontent.com/tensorflow/tfjs-models/master/body-pix/dist/body-pix.min.js
- **크기**: 약 200KB
- **참고**: npm CDN 경로가 변경되어 GitHub에서 직접 다운로드

### 3. DeepLab 모델 (일반 객체 세그멘테이션 - 추천)
- **파일**: `deeplab.min.js`
- **다운로드**: https://unpkg.com/@tensorflow-models/deeplab@0.2.1/dist/deeplab.min.js
- **크기**: 약 17KB (로더만, 실제 모델은 별도 다운로드 필요)
- **모델 파일**: https://github.com/tensorflow/tfjs-models/tree/master/deeplab
- **전체 크기**: 약 5-10MB (모델 포함)

## 📝 업로드 방법

1. **카페24 관리자 페이지 접속**
   - 디자인 > 디자인보관함

2. **폴더 생성**
   - `tfjs` 또는 `ai-libs` 같은 폴더 생성

3. **파일 업로드**
   - 위 파일들을 다운로드 후 업로드

4. **HTML 수정**
   ```html
   <!-- 기존 CDN 경로 주석 처리 -->
   <!-- <script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.10.0/dist/tf.min.js"></script> -->
   
   <!-- 카페24 디자인보관함 경로로 변경 -->
   <script src="https://your-domain.cafe24.com/design/tfjs/tf.min.js"></script>
   <script src="https://your-domain.cafe24.com/design/tfjs/body-pix.min.js"></script>
   ```

## ⚠️ 주의사항

1. **파일 크기 제한**
   - 카페24 디자인보관함의 파일 크기 제한 확인 필요
   - 대용량 모델 파일은 CDN 사용 권장

2. **CORS 설정**
   - 카페24에서 CORS 헤더 설정이 필요할 수 있음
   - 문제 발생 시 카페24 고객센터 문의

3. **캐시 문제**
   - 파일 업데이트 시 브라우저 캐시 삭제 필요
   - 파일명에 버전 번호 추가 권장 (예: `tf.min.v4.10.0.js`)

## 🔄 대안: CDN 사용

카페24 업로드가 어려운 경우:
- jsDelivr CDN 사용 (현재 설정)
- unpkg CDN 사용
- Cloudflare CDN 사용
