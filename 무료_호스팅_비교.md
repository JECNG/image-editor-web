# 무료 Python Flask 백엔드 호스팅 비교

## 🆓 무료 플랜 비교

### 1. Render.com
- **무료 플랜**: ✅ 있음
- **제한사항**:
  - 750시간/월 (약 31일 중 25일 정도)
  - 15분 비활성 시 자동 sleep (첫 요청 시 깨어남, 약 30초 지연)
  - 512MB RAM
  - 무제한 대역폭
- **장점**: GitHub 연동 쉬움, 자동 배포
- **단점**: Sleep 기능으로 첫 요청 느림

### 2. Railway.app
- **무료 플랜**: ✅ 있음 (크레딧 기반)
- **제한사항**:
  - $5 무료 크레딧/월
  - 사용량에 따라 차감 (약 500시간 정도)
  - Sleep 없음 (항상 실행)
- **장점**: Sleep 없음, 빠른 응답
- **단점**: 크레딧 소진 시 중단

### 3. Fly.io
- **무료 플랜**: ✅ 있음
- **제한사항**:
  - 3개 앱 무료
  - 256MB RAM/앱
  - 160GB 대역폭/월
- **장점**: Sleep 없음, 빠름
- **단점**: 설정이 조금 복잡

### 4. PythonAnywhere
- **무료 플랜**: ✅ 있음
- **제한사항**:
  - 1개 웹앱
  - 외부 URL 접근 제한 (무료는 `username.pythonanywhere.com`)
  - CPU 시간 제한
- **장점**: Python 전용, 안정적
- **단점**: 외부 도메인 연결 제한

### 5. Replit
- **무료 플랜**: ✅ 있음
- **제한사항**:
  - Always-on 필요 시 유료
  - 무료는 sleep 가능
- **장점**: 브라우저에서 코딩 가능
- **단점**: Always-on은 유료

## 🏆 추천 순위

- 설정 간단

- 첫 요청 시 30초 지연 (Sleep에서 깨어남)
- 이후 요청은 빠름

### 3순위: **Fly.io** (무료, Sleep 없음)
- 빠르고 안정적
- 설정이 조금 복잡
